# Stellar — 3D 성도 뷰어 와이어프레임

> MVP 범위: 3D 인터랙티브 성도 (AR·센서 제외)

---

## 1. 화면 목록

| 화면 ID | 이름           | 설명                        |
| ------- | -------------- | --------------------------- |
| S-01    | Splash         | 앱 로딩                     |
| S-02    | Permission     | 위치 권한 요청              |
| S-03    | StarMap (메인) | 3D 천구 뷰어                |
| S-04    | Star Detail    | 별 상세 정보 (Bottom Sheet) |
| S-05    | Settings       | 날짜·위치·필터 설정         |

---

## 2. S-01 Splash

```
┌─────────────────────────┐
│                         │
│                         │
│        ★ STELLAR        │
│     (로고 + 앱 이름)     │
│                         │
│    ────────────────     │
│    로딩 프로그레스바      │
│                         │
│                         │
└─────────────────────────┘
```

**동작**

- 앱 진입 시 1.5초 표시 후 자동으로 S-02 또는 S-03 이동
- 위치 권한이 이미 허용된 경우 S-03으로 바로 진입

---

```
┌─────────────────────────┐
│                         │
│       📍                │
│   현재 위치가 필요해요    │
│                         │
│  정확한 밤하늘을 보려면  │

│  사용됩니다.             │
│                         │
│  ┌───────────────────┐  │
│  │   위치 허용하기    │  │
│  └───────────────────┘  │
│                         │
```

## 4. S-03 StarMap (메인 화면)

```
┌─────────────────────────┐
│ ☰            STELLAR  ⚙ │  ← 상단 바 (설정 버튼)
├─────────────────────────┤
│                         │
│   [  3D 천구 뷰         │
│                         │
│     · ·  ✦ · ·  ·      │
│    ·   별자리 선   ·     │
│   ·  (드래그로 회전)  ·  │
│    ·  (핀치로 줌)   ·   │
│     · · ·  ✦ · · ·     │
│                         │
│   ]                     │
│                         │
├─────────────────────────┤
│ 🔍  별자리 검색          │  ← 검색창
├──────────┬──────────────┤
│ 등급 ≤ 3 │  별자리 선 ON │  ← 토글 필터
├──────────┴──────────────┤
│ 📅 2026-06-12  00:00    │  ← 날짜/시간 슬라이더 (탭하면 확장)
└─────────────────────────┘
```

### 3D 뷰 인터랙션

| 제스처           | 동작                   |
| ---------------- | ---------------------- |
| 한 손가락 드래그 | 천구 회전              |
| 두 손가락 핀치   | 줌 인/아웃             |
| 별 탭            | S-04 Bottom Sheet 열기 |
| 더블 탭          | 해당 영역 중앙 정렬    |

### 날짜/시간 슬라이더 (확장 시)

```
┌─────────────────────────┐
│  ◀  2026-06-12  ▶       │
│  ○──────●───────○       │  ← 시간 슬라이더 (0h ~ 24h)
│         23:00           │
│  [현재 시간으로 초기화]  │
└─────────────────────────┘
```

## 5. S-04 Star Detail (Bottom Sheet)

```
┌─────────────────────────┐
│ ──────── (드래그 핸들)   │
│                         │
│  ✦ Sirius               │  ← 별 이름 (국문/영문)
│  큰개자리 α              │  ← 소속 별자리
│                         │
│  ┌──────┬──────┬──────┐ │
│  │ 등급  │ 거리  │ 온도 │ │
│  │ -1.46 │ 8.6ly│6700K│ │
│  └──────┴──────┴──────┘ │
│                         │
│  적경  06h 45m 08s      │
│  적위  -16° 42' 58"     │
│                         │
│  [  이 별자리 전체 보기  ]│  ← 해당 별자리 하이라이트
│                         │
└─────────────────────────┘
```

**동작**

- 위로 스와이프 → 전체 화면 확장
- 아래로 스와이프 → 닫기
- "이 별자리 전체 보기" → 3D 뷰에서 해당 별자리 하이라이트 + 자동 회전

## 9. 데이터 흐름

```
expo-location (GPS)
        │
        ▼
astronomy-engine
(적경/적위 → 방위각/고도 계산)
        │
        ▼
Three.js Scene
(Hipparcos 카탈로그 별 좌표 매핑)
        │
        ▼
StarMapCanvas 렌더링
```
